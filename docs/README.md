# Documentation Map

1. [Project README](../README.md)
   Public-facing overview, architecture summary, setup flow, and roadmap.

2. [Architecture Notes](./architecture.md)
   Focused explanation of semantic grounding, execution guardrails, local-first analytics, and trust surfaces.

3. [Platform Overview](../Design%20Doc/platform_overview.md)
   Broader product framing, platform milestones, and implementation phases.

4. [Full Design Specification](../Design%20Doc/HELIOS_Full_Design_Specification.md)
